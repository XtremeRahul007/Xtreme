Volumetric Fog (URP)

Volumetric Fog (URP) is a feature that provides a volumetric fog effect in Unity's Universal Render Pipeline (URP). It allows you to create a realistic and immersive fog effect that can be used in your 3D projects. The fog effect can be applied to various objects, including terrain, buildings, and character models, and can be controlled by various settings, such as density, color, and gradient.
